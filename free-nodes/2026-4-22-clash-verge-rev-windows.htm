一、Quantumult简介与基本概念

什么是Quantumult？

Quantumult是一款面向iOS平台的网络代理软件，支持多协议代理服务，帮助用户突破网络限制，保护上网隐私。其强大的自定义参数设定和丰富的功能模块，使其不仅适合普通用户，也满足高级用户的个性化需求。

基础概念解析

  • 代理（Proxy）：通过代理服务器转发网络请求，隐藏用户真实IP地址，绕过网络限制。

  • 规则（Rules）：用于判断哪些网络流量走代理，哪些流量直连，提升访问效率。

  • 配置文件（Configuration File）：保存代理服务器、规则、脚本等全部设定，是Quantumult运行的核心。

理解这几个基础概念，有助于您更好地掌握Quantumult的配置技巧。

二、Quantumult的安装流程

Quantumult可通过苹果App Store下载，安装过程简单：

  1. 打开App Store，搜索“Quantumult”。

  2. 点击下载并安装，等待完成。

  3. 启动应用，进入主界面即可开始配置。

安装简便，用户能够快速进入配置环节。

三、Quantumult的基本参数设定

1. 代理服务器添加与管理

Quantumult支持多种代理类型，包括HTTP、HTTPS、SOCKS5等，用户可以灵活添加：

  • 路径：进入“设置” → “代理” → “添加代理”。

  • 配置代理信息：填写代理服务器地址、端口号、用户名和密码（如需）。

  • 选择代理类型：根据代理服务选择对应协议。

添加完成后，代理列表会自动更新，方便管理和切换。

2. 规则的设置与优化

规则决定哪些流量通过代理，哪些流量直连：

  • 添加规则：在“设置” → “规则”中添加自定义规则。

  • 规则格式：支持域名匹配、IP段匹配等多种格式。

  • 优先级管理：规则顺序影响匹配效果，优先级高的规则优先应用。

合理配置规则，既能保障访问速度，又能确保关键流量通过代理。

3. 配置文件的导入

配置文件集中保存所有代理和规则信息，导入方式多样：

  • 本地导入：在“设置” → “配置文件” → “导入”中选择本地文件。

  • 在线导入：通过配置文件URL直接导入，方便快捷。

导入后，Quantumult自动应用配置，极大简化设置流程。

四、Quantumult高级参数设定

1. 脚本的使用

Quantumult支持多种脚本语言，用户可以通过脚本实现高级功能：

  • 添加脚本：进入“设置” → “脚本”，导入或编写自定义脚本。

  • 常用脚本应用：实现广告屏蔽、智能分流、自动切换节点等。

  • 语言支持：JavaScript等主流脚本语言，扩展功能强大。

脚本极大提升了Quantumult的灵活性和自动化水平。

2. 订阅功能

订阅功能方便用户批量管理代理服务器：

  • 添加订阅：在“设置” → “订阅”中添加订阅链接。

  • 自动更新：设置自动刷新间隔，保证订阅节点及时更新。

  • 多订阅支持：可添加多个订阅，满足不同需求。

订阅使节点管理更便捷，维护代理稳定性更简单。

1. 流量监控

Quantumult内置流量监控模块，帮助用户掌握数据使用情况：

FAQ

Mellow TUN 模式与 SOCKS5 代理有什么区别？
TUN 模式拦截系统 IP 层流量，实现全局代理；SOCKS5 代理通常只影响指定应用。TUN 更适合全局加速，如游戏、视频和浏览器流量，提高网络稳定性和覆盖范围。
Clash 如何防止规则文件冲突导致分流异常？
合理组织规则顺序，将更具体的规则放在前面，通配规则放在末尾；使用 rule-providers 分离不同来源的规则，避免重复条目，并在变更后使用日志或 debug 模式检验分流是否按预期生效。
Sing-box 如何保证规则订阅的实时更新？
Sing-box 支持远程规则订阅 URL，可设置定期自动更新频率，如每小时或每天。自动更新保证节点和规则保持最新，减少手动维护，提高访问速度和分流效率。
WinXray 多策略组如何管理大量节点？
多策略组可将节点按地区、延迟或用途分类，并结合规则文件自动分流流量。这样可快速选择最优节点，提高访问速度和稳定性，同时优化节点管理和维护效率。
V2Ray 配置中如何避免端口扫描暴露服务？
避免使用常见端口（如 80、443）直接暴露 V2Ray 服务；若必须使用 443，建议通过 Web 服务器反代或使用 CDN；同时启用 fail2ban 与 iptables 限制异常连接频率，减少被扫描发现的风险。
VLESS + XTLS 模式在移动网络下表现如何？
XTLS 降低加密开销，减少握手次数，适合高延迟或不稳定的移动网络。结合策略组和多出口节点，可提升访问速度和连接稳定性，确保用户体验流畅。
Sing-box 如何实现精确端口分流？
Sing-box 支持按端口匹配流量，将不同类型应用流量分配到指定节点。结合策略组，可提高访问速度和网络稳定性，实现精准分流和优化用户体验。
Matsuri 可以用于 Android 吗？
Matsuri 主要针对桌面平台开发，目前并未推出 Android 版本。但可使用其配置文件在兼容的客户端（如 Nekobox 或 Clash Android）中加载以获得类似体验。
Shadowrocket 节点延迟排序如何提高稳定性？
Shadowrocket 测试节点延迟和丢包率，将低延迟节点优先使用。结合策略组和规则文件，可保证网页、视频和游戏流量通过最优节点，提高连接稳定性和访问速度。
Sing-box 支持哪些类型的分流规则？
Sing-box 支持域名、IP、端口、协议类型、用户标签等多种分流规则。结合策略组，可实现精细化流量管理，满足视频加速、广告屏蔽和游戏优化等多样化需求。
